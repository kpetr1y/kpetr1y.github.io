Two-Tier Client–Server Database Application



DESCRIPTION

Developed a two-tier client–server database application using Java Swing for the GUI and JDBC for MySQL connectivity, enabling efficient record management with built-in error handling and data validation for a better user experience.


SKILLS DEVELOPED

  • Java Swing for Graphical User Interface Development
  • JDBC for Database Interaction
  • MySQL Database Design and Management
  • Implementation of CRUD Operations
  • Error Handling and Data Validation Techniques
  • Client-Server Architecture Principles
