Three-Tier Distributed Web Application



DESCRIPTION

Developed a secure three-tier web application enabling browser-based interaction with a relational database, including user authentication and role-based access control.Implemented server-side business logic to enforce application rules, route users by authorization level, and generate database-driven reports.


SKILLS DEVELOPED

  • Java Servlet Programming
  • Relational Database Management with MySQL
  • Web Server Configuration using Apache Tomcat
  • HTML, CSS, and JavaScript for Front-End Development
  • User Authentication and Authorization Techniques
  • Implementation of Business Logic in a Multi-Tier Architecture
